# Portfolio Optimization using HHL Quantum Algorithm

## Overview
This notebook demonstrates how to solve portfolio optimization problems using the **HHL quantum algorithm**, seamlessly combining classical financial theory with cutting-edge quantum computing. The project explores how quantum computers can efficiently solve linear systems of equations to optimize investment portfolios.

## What's Included
- Implementation of the HHL (Harrow-Hassidim-Lloyd) quantum algorithm
- Classical portfolio optimization theory and principles
- Quantum circuit design and execution
- Comparative analysis between classical and quantum approaches
- Real-world portfolio optimization examples

## Key Features
- Efficient quantum solutions for portfolio optimization
- Integration of quantum computing with financial analysis
- Detailed explanations of quantum algorithms applied to finance
- Practical implementation using quantum computing frameworks
